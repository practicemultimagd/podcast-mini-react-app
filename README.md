# Podcaster mini app

The project is separated into two sections, "client" and "server".
The server part is not implemented properly. This is only to simulate 
clean responses.

## Dependences
node-js version v19.4.0

## How to use it

### develop mode
```
cd server
npm i
npm run ts:dev
cd ../client
npm i
npm run dev
```
### production preview mode
```
cd server
npm i
npm run ts:dev
cd ../client
npm i
npm run preview
```
